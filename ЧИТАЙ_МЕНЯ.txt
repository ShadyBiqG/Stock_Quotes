1. ПОЛУЧИТЬ НОВЫЙ API КЛЮЧ

   Откройте: https://openrouter.ai/keys
   
   • Войдите в аккаунт (или зарегистрируйтесь)
   • Нажмите "Create API Key"
   • Дайте имя ключу (например: "Stock Analyzer")
   • СКОПИРУЙТЕ ключ (показывается только один раз!)
   • Ключ начинается с: sk-or-v1-


2. ПОПОЛНИТЬ БАЛАНС

   Откройте: https://openrouter.ai/credits
   
   • Минимум: $5
   • Рекомендуется: $10-20
   • Оплата: карта, PayPal, crypto


3. ОБНОВИТЬ КОНФИГУРАЦИЮ

   Откройте файл: config.yaml
   
   Найдите строку:
   api_key: "sk-or-v1-старый-ключ"
   
   Замените на:
   api_key: "sk-or-v1-ваш-новый-ключ"
   
   Сохраните файл (Ctrl+S)


4. ПРОВЕРИТЬ КЛЮЧ

   Запустите: bin\test_api_key.bat
   
   Должно быть:
   ✅ API КЛЮЧ РАБОТАЕТ!


5. ЗАПУСТИТЬ ПРИЛОЖЕНИЕ

   Веб-интерфейс: bin\start_web.bat
   CLI версия:    bin\start.bat
