🚀 ЗАПУСК (ВЫБЕРИТЕ ОДИН ВАРИАНТ):

1️⃣ ЕСЛИ УЖЕ НАСТРОЕНО:
   
   Запустите:  quick_start.bat  или  start_web.bat
   
   Откроется: http://localhost:8501

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2️⃣ ЕСЛИ ПЕРВЫЙ ЗАПУСК:

   Шаг 1: Установка
   ═══════════════════
   Запустите:  setup.bat
   
   Что сделает:
   - Проверит Python
   - Создаст виртуальное окружение
   - Установит все зависимости
   - Проверит структуру
   
   Шаг 2: API ключ
   ═══════════════
   1. Откройте config.yaml
   2. Найдите: api_key: "your-openrouter-api-key-here"
   3. Замените на ваш ключ с https://openrouter.ai/
   4. Пополните баланс ($10-20 для начала)
   
   Шаг 3: Запуск
   ══════════════
   Запустите:  start_web.bat
   
   Откроется: http://localhost:8501

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 ДОСТУП С ТЕЛЕФОНА:

После запуска start_web.bat вы увидите адреса:

  💻 Локально:       http://localhost:8501
  📱 В сети Wi-Fi:   http://192.168.X.X:8501
  
Откройте второй адрес на телефоне (та же Wi-Fi сеть).

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 ПЕРВЫЕ ШАГИ:

1. Добавьте компании
   ══════════════════
   Настройки → Конфигурация → Управление компаниями
   Введите только Ticker (например: AAPL, GOOGL, MSFT)
   Нажмите "Добавить" - готово!

2. Подготовьте Excel
   ═══════════════════
   Файл: Stock quotes.xlsx
   Колонки: Ticker, Price, Change, Volume
   
   Пример:
   Ticker | Price  | Change | Volume
   AAPL   | 178.25 | 2.5    | 55000000
   GOOGL  | 142.10 | -1.2   | 28000000

3. Запустите анализ
   ═════════════════
   Обзор → Запустить анализ
   Дождитесь завершения
   Результаты появятся автоматически

4. Экспортируйте
   ══════════════
   Обзор → Экспорт в Excel
   Скачайте форматированный отчет

💡 ВАЖНО:

✅ Теперь достаточно только Ticker при добавлении компаний
✅ Информация получается автоматически (Yahoo Finance → LLM)
✅ Кэширование на 30 дней (быстрота + экономия)
✅ Fallback на LLM если Yahoo Finance недоступен

❌ НЕ нужно вручную заполнять описание, сектор и т.д.
❌ НЕ нужно беспокоиться если Yahoo Finance не работает

🐛 РЕШЕНИЕ ПРОБЛЕМ:

❌ API ключ не настроен
   → Откройте config.yaml и настройте ключ

❌ Файл Stock quotes.xlsx не найден
   → Создайте Excel файл или загрузите через веб-интерфейс

❌ Yahoo Finance API не работает
   → Это нормально! LLM fallback сработает автоматически

❌ Зависимости не установлены
   → Запустите: setup.bat
