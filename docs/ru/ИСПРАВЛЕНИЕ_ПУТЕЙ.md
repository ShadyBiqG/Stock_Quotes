# Исправление: Файл Stock quotes.xlsx не найден

## Проблема
После реструктуризации проекта возникала ошибка:
```
❌ Файл Stock quotes.xlsx не найден!
```

## Причина
Файл был перемещен из корня проекта в `data/samples/`, но код продолжал искать его в корне.

## Решение ✅

### 1. Добавлен параметр в конфигурацию

В `config.yaml` и `config.example.yaml` добавлен параметр:
```yaml
# Путь к файлу с котировками
input:
  excel_file: "data/samples/Stock quotes.xlsx"
```

### 2. Обновлен код

Все файлы обновлены для использования нового пути:
- ✅ `main.py` - читает путь из конфига
- ✅ `src/data_loader.py` - путь обновлен
- ✅ `src/analyzer.py` - путь обновлен
- ✅ `src/dashboards/overview.py` - путь обновлен
- ✅ `src/dashboards/settings.py` - путь обновлен (3 места)

### 3. Обновлена документация
- ✅ `README.md` - инструкции по настройке пути
- ✅ `PATHS_UPDATED.md` - полное описание изменений (на английском)
- ✅ `docs/ru/ИСПРАВЛЕНИЕ_ПУТЕЙ.md` - это руководство

## Как использовать

### Вариант 1: Стандартный (рекомендуется)

Оставьте настройки по умолчанию в `config.yaml`:
```yaml
input:
  excel_file: "data/samples/Stock quotes.xlsx"
```

Файлы-образцы уже находятся в `data/samples/`.

### Вариант 2: Свой путь

Укажите свой путь к файлу:
```yaml
input:
  excel_file: "путь/к/вашему/файлу.xlsx"
```

### Вариант 3: Через веб-интерфейс

1. Запустите `bin\start_web.bat`
2. Перейдите в **Настройки → Анализ**
3. Загрузите свой Excel файл
4. Файл автоматически сохранится в правильное место

## Проверка

Убедитесь, что файл существует:
```bash
dir data\samples\Stock quotes.xlsx
```

Должно показать файл размером ~10-50 КБ.

## Если проблема осталась

1. **Проверьте config.yaml:**
   ```bash
   type config.yaml | findstr "excel_file"
   ```
   
   Должно быть:
   ```yaml
   excel_file: "data/samples/Stock quotes.xlsx"
   ```

2. **Проверьте структуру папок:**
   ```bash
   dir data\samples
   ```
   
   Должна быть папка `samples` с файлами Excel.

3. **Создайте папку, если нет:**
   ```bash
   mkdir data\samples
   ```

4. **Скопируйте файл:**
   Если у вас есть `Stock quotes.xlsx` в другом месте:
   ```bash
   copy "Stock quotes.xlsx" "data\samples\"
   ```

## Запуск приложения

После исправления:
```bash
# CLI
bin\start.bat

# Веб-интерфейс
bin\start_web.bat
```

Приложение должно успешно найти и загрузить файл! ✅
